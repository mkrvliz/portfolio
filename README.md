# mkrvliz.github.io

портфолио лизы макаровой — веб-дизайнера и вайбкодера.

## стек

статический сайт без сборки. чистые html / css / vanilla js. шрифт — geist mono с google fonts. хостинг — github pages с раздачей из `main`.

## структура

```
.
├── index.html                       — главная (hero, обо мне, портфолио, услуги, этапы, контакты)
├── project-02.html                  — кейс "Юность"
├── project-oh-keramika/index.html   — кейс "ох! керамика"
├── css/
│   ├── common.css                   — шапка, футер, прелоадер, бургер-меню, кнопка "наверх"
│   ├── home.css                     — стили главной
│   └── project.css                  — стили страниц кейсов (галерея, слайдер, лайтбокс)
├── js/
│   ├── common.js                    — прелоадер, плавный скролл, бургер, кнопка "наверх"
│   ├── home.js                      — fade-in блоков, аккордеон услуг, фон с анимацией символов
│   └── project.js                   — fade-in скринов, слайдеры этапов, лайтбокс с лупой
└── images/                          — изображения проектов
```

## особенности

- **прелоадер** показывается только при первом заходе в сессию (через `sessionStorage`).
- **плавный скролл** (lerp) — только на десктопе (`innerWidth > 1024`), на мобильном нативный.
- **typewriter** анимация — только на заголовке hero, чтобы не обрезать буквы на iOS safari.
- **fade-in** анимация блоков — единая фабрика `fadeIn(selector, stagger, reveal)` в `home.js`.
- **слайдер** на страницах кейсов — кастомный, без библиотек. на мобильном автоматически конвертирует `.stage-screens` в слайдеры.
- **лайтбокс** с лупой при наведении (только на устройствах без тача).

## разработка

любой статический сервер из корня репозитория, например:

```sh
python -m http.server 3000
```
